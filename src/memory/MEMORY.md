# Abraxas Memory

## Always-On Rules
- Capture casual user ideas into `src/memory/braindump.md` with timestamp and tags.
- Do not ask follow-up questions for braindump capture; store first, clarify later only if needed.
- When current discussion semantically matches previous dumps, proactively surface relevant entries.
- Keep this file concise. Put raw or noisy notes into `src/memory/braindump.md`.

## Memory Layers
- `src/memory/MEMORY.md`: distilled durable memory injected on startup.
- `src/memory/braindump.md`: raw idea inbox with tags.
- `src/memory/daily/YYYY-MM-DD.md`: daily structured sync logs.
- `src/memory/mission-log.md`: periodic conversion from braindump to actionable backlog.

## Scheduler Defaults
- Daily memory sync timezone: `Asia/Shanghai`
- Daily memory sync time: `02:00`
