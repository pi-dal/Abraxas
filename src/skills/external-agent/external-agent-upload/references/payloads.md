# Upload Payload Shapes

Upload into existing project:

```json
{
  "projectId": "proj-1",
  "filePath": "assets/file.pdf",
  "contentBase64": "...",
  "overwrite": true
}
```

Import as new project:

```json
{
  "fileBase64": "...",
  "fileName": "paper.zip",
  "ownerId": "user-1",
  "name": "Paper Import",
  "description": "optional"
}
```
