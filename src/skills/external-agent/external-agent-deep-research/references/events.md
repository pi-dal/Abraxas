# Deep Research SSE Event Hints

Common event names from `/api/deep-research/stream`:

- `progress`
- `search_start`
- `search_result`
- `analysis`
- `iteration`
- `outline_start`
- `outline_done`
- `section_start`
- `section_chunk`
- `section_done`
- `report_start`
- `report_chunk`
- `report_done`
- `done`

The `done` event typically contains final report data and references.
